# 🚀 YouTube Master Command Center v6.0

Welcome to the **Ultimate YouTube Creator Suite**. This is a professional-grade Python Flask application designed to automate the most boring parts of YouTube content creation. 

## 🛠 Features Included:
- 🔥 **Viral Title Generator:** Get high CTR titles for your videos.
- 🏷️ **SEO Tags & Keywords:** Generate algorithm-friendly tags.
- #️⃣ **Hashtag Generator:** Instant viral hashtags for Shorts and Long-form.
- 🎬 **Full Video Scripting:** Hook, Intro, Body, and Outro structure.
- 📝 **Optimized Descriptions:** Ready-to-paste SEO descriptions with timestamps.

## 📦 Tech Stack:
- **Language:** Python 3.x
- **Framework:** Flask
- **Server:** Gunicorn (for 24/7 Hosting)
- **UI:** Tailwind CSS (Elite Studio Dark Theme)

## 🚀 Deployment on Render.org

1. **Connect GitHub:** Link this repository to your Render account.
2. **Settings:**
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
3. **Environment:** Render will automatically assign a public URL for your APK.

## 📱 Mobile Usage
This dashboard is fully responsive. You can convert the Render URL into an APK using any web-to-apk tool and it will work **without Termux** since it's hosted on the cloud.
